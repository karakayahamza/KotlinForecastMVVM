# KotlinForecastMVVM

KotlinForecastMVVM adlı uygulama, Kotlin programlama dilini kullanarak geliştirilen ve Android platformunda çalışan bir hava durumu uygulamasıdır. Bu uygulama, kullanıcıların güncel hava durumu bilgilerini takip edebilmeleri, şehirler ekleyip çıkarabilmeleri ve 5 günlük sıcaklık tahminlerini görüntüleyebilmeleri için tasarlanmıştır.

## Kullanılan Teknolojiler

- Kotlin
- MVVM Mimarisi
- ViewPager
- Retrofit ve RxJava
- OpenWeatherAPI
- Google Maps API
- Navigation Fragment

## Özellikler

- Güncel konumun hava durumunu anlık olarak takip edebilme
- Şehirler ekleyebilme ve çıkarabilme
-Güncel sıcaklık, nem, rüzgar hızı ve basınç gibi ayrıntıları öğrenebilme
